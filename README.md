# 알고리즘 오답 노트

## 1. 투포인터, 슬라이딩 윈도우 문제
**문제 설명**
---
1110001110111 과 같은 배열이 주어지고, 0을 채울 수 있는 횟수(n)가 정해져 있음. 연속되는 최대 1의 크기를 구하는 문제.  
n의 크기는 최대 3 * 10^5.

**Point**
---
- 조합으로 풀 수 있으나, n의 크기로 인해서 투포인터, 슬라이딩 윈도우로 풀어야하는 문제  
  
출처 : 현대카드

## 2. 완전탐색
**문제 설명**
---
2개의 가스통을 채울 수 있는 3가지 조건이 있고, 2개의 가스통에 채워질 수 있는 물의 양을 오름차순으로 정렬하여 배열을 리턴해야함.  

**Point**
---
처음에 문제를 이해못했으나, 6가지 조건에 따라 완전 탐색을 통해 가스통에 채워지는 물의 양을 기록하면 된다(각 가스통마다 3가지 조건)  
  
출처 : 현대카드
